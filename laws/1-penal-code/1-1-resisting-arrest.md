# \[1] 1 - Resisting Arrest



{% hint style="danger" %}
**Felony**

**Sentence:** 200 Seconds \
**Fine:** $1,000
{% endhint %}

{% hint style="warning" %}
May not stack with Disobeying a Police Officer
{% endhint %}

**Examples:**

**1)** Struggling with police officers as they try to apply handcuffs\
**2)** Running away from police on foot
