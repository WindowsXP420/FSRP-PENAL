# \[3] 29 - Hit and Run

Causing a traffic collision and failing to stop and identify oneself afterwards.&#x20;

{% hint style="danger" %}
**Misdemeanor**\
\
**Sentence:** 100 Seconds\
**Fine:** $1,000
{% endhint %}

{% hint style="danger" %}

**Fine:** $2,000
{% endhint %}
